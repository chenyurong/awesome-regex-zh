# 强大的正则表达式

在 Github 上有一个[Awesome Regex](https://github.com/aloisdg/awesome-regex)，收集了关于正则表达式的表达式库、工具、框架、软件等。本项目在其基础上，根据中国的实际情况，整理一份适用于中国国情的正则表达式库。当需要使用正则表达式时，可以直接到这里搜索，直接使用。
